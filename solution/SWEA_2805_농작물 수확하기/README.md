# 2805. 농작물 수확하기

### solution
* 이중 for문을 사용하여 해결
* x, y변수를 N/2로 초기화
* x <= j <= y로 열의 범위를 지정해 주어 범위 내의 숫자만 add
* x는 감소, y는 증가하다가 x가 0이 되고 y가 N-1이 되었을 때 체크
* 이제 부터는 x는 증가, y는 감소하며 add 수행
