# 18352. 특정 거리의 도시 찾기

### classification
* Graph theory
* Graph search
* BFS
* Dijkstra algorithm

### solution
* 연결 리스트를 이용하여 인접 리스트 구현
* 우선순위 큐를 이용하여 다익스트라를 구현
* X부터 시작해 다른 노드들 까지의 최단 거리를 구함
* 최단 거리가 K인 노드의 개수를 셈

### remind
* Dijkstra algorithm
* Priority queue
