# 2116. 주사위 쌓기

### classification
* Implementation
* Brute force

### solution
* 주사위의 밑면과 윗면이 설정되었을 때 나올 수 있는 옆면 중 가장 큰 수 선택
* 윗면과 밑면 중 큰 수가 6이 아니면 옆 면중 가장 큰 수는 6
* 윗면 또는 밑면이 6일 때 다른 한 쪽면이 5가 아니면 옆면은 5
* 윗면 또는 밑면이 6일 때 다른 한 쪽면이 5이면 옆면은 4
