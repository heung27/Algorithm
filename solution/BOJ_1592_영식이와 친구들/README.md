# 1592. 영식이와 친구들

### classification
* Math
* Implementation
* Simulation

### solution
* N크기의 배열을 선언하여 각 사람이 공을 받은 횟수 카운트
* 해당 사람이 공을 받은 횟수가 홀수일 때 인덱스에 +L을 해주어 시계방향으로 이동
* 해당 사람이 공을 받은 횟수가 짝수일 때 인덱스에 -L을 해주어 반시계방향으로 이동
* 공을 받은 횟수가 M이 되면 종료하고 카운트한 총 던진 횟수 출력
