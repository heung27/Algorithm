# 17140. 이차원 배열과 연산

### classification
* Implementation
* Simulation

### solution
* temp 배열에 각 행 또는 열의 숫자들을 삽입하고 101 크기의 num 배열을 이용해 각 숫자의 빈도를 계산
* temp를 빈도수를 기준으로 정렬하고 순서대로 숫자와 빈도수 저장
* 가장 큰 행 또는 열을 구하여 배열 확장
