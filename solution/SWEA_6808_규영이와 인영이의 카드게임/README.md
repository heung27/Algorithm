# 6808. 규영이와 인영이의 카드게임

### solution
* 입력받은 규영이의 카드 숫자 저장
* 규영이의 카드 숫자를 제외한 나머지 9개의 카드가 인영이의 카드
* NextPermutation을 구현하여 인영이 카드의 모든 순열을 탐색
* 모든 순열에 대해 규영이와 인영이의 총점을 계산하여 규영이가 이기고 지는 경우 count
