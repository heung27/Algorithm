# 1793. 타일링

### classification
* Dynamic programming
* Arbitrary precision / Big integers

### solution
* 숫자가 매우 커져 자바에서는 BigInteger를 사용해야함
* 2xN 크기의 타일을 채우는 방법은 2x(N-2) 크기의 타일을 채우는 방법 * 2 + 2x(N-1) 크기의 타일을 채우는 방법
