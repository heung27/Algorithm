# 1233. 사칙연산 유효성 검사

### solution
* 이진트리로 표현된 사칙연산 식이 계산이 가능한 식인지 판별
* 계산이 가능하기 위해서는 단말 노드들만이 숫자여야 함
* 트리의 노드 정보를 입력받으면서 노드가 연산자가 아닌 숫자일 때 자식노드가 존재하면 계산이 불가능한 트리라고 판단
* 모든 입력이 성공적으로 전부 들어오면 계산이 가능한 트리
