# 1230. 암호문3

### solution
* 입력 데이터를 리스트에 전부 add
* 문자 I를 입력 받으면 x의 위치에 y개의 문자열 삽입
* 문자 D를 입력 받으면 x의 위치의 문자열 삭제
* 문자 A를 입력 받으면 리스트의 맨 뒤에 y개의 문자열 삽입
* 문자열이 삽입,삭제 될 때 마다 x++하여 인덱스 조정
