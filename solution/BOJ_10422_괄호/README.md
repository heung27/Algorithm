# 10422. 괄호

### classification
* Math
* Dynamic programming
* Combinatoric

### solution
* 카탈란 수를 활용하는 문제
* 카탈란 수는 조합론에서 빈번하게 나타나는 수열
* 1, 1, 2, 5, 14, 42, 132, 429, ... 순으로 증가하는 수열
* 여기에 전부 설명하기엔 무리가 있어 자세한 내용은 인터넷 참조
