# 16926. 배열 돌리기1

### classification
* Implementation

### solution
* 방향 배열을 만들어 회전 방향의 index를 계산
* 맨 바깥의 테두리부터 회전 진행
* N이 M보다 작을 때 N/2번째 테두리까지 회전 진행
* N이 M보다 클 때 M/2번째 테두리까지 회전 진행
* 맨 처음의 값을 temp에 저장하고 한 칸씩 값들이 이동, 마지막 칸에 temp 저장
