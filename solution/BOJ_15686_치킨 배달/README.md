# 15686. 치킨 배달

### classification
* Implementation
* Brute force algorithm

### solution
* 집의 좌표를 저장하는 list를 생성
* 치킨집의 좌표를 저장하는 list 생성
* 치킨집을 M 만큼 선택할 수 있는 모든 조합을 탐색
* 치킨집을 M개 선택했을 때 각 집에서 치킨집 까지의 거리를 계산하여 최솟값 저장
* 모든 경우의 수에 동일한 연산 반복
