# 14890. 경사로

### classification
* Implementation

### solution
* 입력되는 지도의 정보를 배열에 저장
* 오른쪽 방향으로 한번, 아래 방향으로 한번 모든 행과 열 탐색
* 탐색을 진행하면서 [k-1] == [k]+1이면 진행방향으로 쭉 이동하면서 [k]와 같은 수가 연속으로 몇 개가 있는지 셈
* 그 값이 L과 같으면 반대로 돌아가면서 check배열에 체크해줌
* 탐색을 진행하면서 [k-1]+1 == [k]이면 반대방향으로 쭉 이동하면서 [k-1]와 같은 수가 연속으로 몇 개가 있는지 셈
* 그 값이 L과 같으면 다시 진행방향으로 돌아가면서 check배열에 체크해줌
* [k-1] == [k]+1도 아니고 [k-1]+1 == [k]도 아니면 break
* 행 또는 열이 중간에 break되지 않고 N-1까지가서 반복문이 종료되면 count 증가
