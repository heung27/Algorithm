# 4991. 로봇 청소기

### classification
* Graph theory
* Graph search
* Brute force
* BFS
* Bitmask

### solution
* 로봇청소기의 좌표와 더러운 칸의 좌표를 저장
* 각 좌표에서 BFS를 통해 좌표들 사이의 거리를 계산
* 로봇청소기의 좌표부터 시작해서 더러운 칸들을 순회하는 모든 경우의 수를 탐색하여 최솟값을 구함

### remind
* BFS
* Brute force
